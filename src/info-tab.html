Quests

Use quests to reward players for completing tasks on your server automatically.

Setup notes

  • You can change quest codes without letting users complete the quest again.
  • If you re-order quests, users that have already completed a quest that they now cannot access will still have completed the quest.
  • You can technically have multiple quests with the same completion code, but this will be confusing to both you and your users and is not recommended.

Levels

When users gain enough XP to get a level, the level up message will be sent. If you change how much XP users need to achieve a level, players levels will change to reflect the new requirement. Each level's XP field is how much XP is required to level up from the previous level.

Commands

  • /QUESTS - List up to 5 incomplete quests
  • /QUEST <title> - Get details about a quest. If no title is provided, this will act like /QUESTS
  • /CODE <code> - Claim XP for completing a quest
  • /LEVEL - Check your current level
  • /LEVELTOP - List the top 10 players with the most XP
  • /XP <name> <amount> (Quest Master only) - Add or remove XP from a player
  • /LEVEL <name> (Quest Master only) - Check the current level of a player
